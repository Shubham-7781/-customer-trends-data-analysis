# 📊 Customer Shopping Trends Analytics

> An end-to-end Data Analytics project that transforms raw retail shopping data into actionable business insights using **Python, SQL, and Power BI**.

---

## 📖 Project Summary

Understanding customer purchasing behavior is essential for improving business performance. This project demonstrates a complete analytics workflow, starting with raw customer transaction data and ending with an interactive Power BI dashboard that supports business decision-making.

The project covers data preparation, exploratory analysis, SQL-based business queries, dashboard creation, and reporting, closely following the workflow used by data analysts in real organizations.

---

## 🎯 Objectives

* Clean and prepare retail customer data
* Explore customer purchasing patterns
* Perform business-oriented SQL analysis
* Build an interactive Power BI dashboard
* Generate insights that support business decisions

## 🔍 Project Workflow

### 1️⃣ Data Collection

* Load retail customer dataset
* Inspect data quality
* Identify missing values and inconsistencies

### 2️⃣ Data Cleaning

* Handle null values
* Remove duplicates
* Correct data types
* Prepare analytical dataset

### 3️⃣ Exploratory Data Analysis

* Customer demographics
* Purchase distribution
* Shopping frequency
* Category analysis
* Correlation analysis

### 4️⃣ SQL Analysis

Business questions explored include:

* Which customer segment contributes the highest revenue?
* Which product categories are most popular?
* How do seasonal purchases vary?
* Which payment methods are preferred?
* What are the spending trends across age groups?

### 5️⃣ Dashboard Development

Interactive Power BI dashboard featuring:

* KPI Cards
* Sales Overview
* Customer Segmentation
* Category Performance
* Purchase Trends
* Regional Analysis
* Interactive Filters
